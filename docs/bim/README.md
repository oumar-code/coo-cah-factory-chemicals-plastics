# Plastics — BIM / 3D Model Stub Package

> **Gate 3 Artifact (Digital Twin Spatial Readiness)**

This folder provides the minimum BIM/3D stub artifacts required before a full IFC/CAD model is available.

## File Index

| File | Purpose |
|------|---------|
| [`zone-boundaries.md`](./zone-boundaries.md) | Canonical zone boundaries and coordinate frames |
| [`asset-anchors.md`](./asset-anchors.md) | Canonical asset anchor points for DT placement |

## Governance

- Zone IDs MUST match [`floor-plan.md`](../../floor-plan.md).
- Asset IDs MUST match [`digital-twin.md`](../../digital-twin.md).
- Coordinate system naming MUST be consistent across all files in this folder.
